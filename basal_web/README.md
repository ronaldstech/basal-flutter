# Basal App Landing Page

This folder contains a premium, single-page landing page for the Basal music streaming app.

## Contents
- `index.html`: Main landing page with hero, features, and CTA sections.
- `download.html`: Dedicated download page for the Android APK.
- `styles.css`: Modern, dark-mode CSS with glassmorphism and Outfit typography.
- `assets/mockup.png`: High-fidelity 3D mockup of the Basal app.

## How to Use
1. Copy this entire `basal_web` folder to your web server.
2. Ensure the APK file built in `build/app/outputs/flutter-apk/app-release.apk` is also accessible if you want to link it directly.
3. Simply open `index.html` in any browser to preview.

## Customization
You can easily change colors or text in `styles.css` and the HTML files to match your brand updates.
